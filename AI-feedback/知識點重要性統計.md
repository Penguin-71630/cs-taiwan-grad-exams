# 計系考古題知識點帶權重要性統計

## 統計方法

掃過 7 校（台大、交大、清大、成大、中央、中正、師大）108–114 年計系考古題的 AI-feedback 題解摘要，
統計每個知識點與多少題目相關，並依學校重要性加權：

| 學校 | 權重 |
|------|------|
| 台大 | 10 |
| 交大 | 9 |
| 清大 | 5 |
| 成大 | 3 |
| 中央 | 1 |
| 中正 | 1 |
| 師大 | 1 |

**星等切法**：0 分 = 0 星、1–14 = ✦、15–39 = ✦✦、40–99 = ✦✦✦、100–199 = ✦✦✦✦、200+ = ✦✦✦✦✦
