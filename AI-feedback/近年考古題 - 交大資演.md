### Problem 4 (Part II)

This problem is worth 6 points on red-black tree node bounds.

- For black height h: N_min(h) = 2^h - 1 (all-black complete tree of height h). N_max(h) = 2^(2h) - 1 = 4^h - 1 (alternating red-black gives maximum nodes, doubling the actual tree height to 2h).

- Key concept: red-black tree size bounds — black height constrains minimum (all-black paths) and maximum (maximum red interleaving) node counts.

### Problem 7 (Part II)

This problem is worth 10 points on Tower of Hanoi time complexity.

- bar(N, X, Y, Z) calls foo(N-1) twice + O(1) work. Recurrence: T(N) = 2T(N-1) + O(1).
- Solution by substitution: T(N) = 2T(N-1)+1 = 2(2T(N-2)+1)+1 = 4T(N-2)+3 = ... = 2^(N-1)T(1) + 2^(N-1)-1 = O(2^N).
- Key concept: Tower of Hanoi recurrence — the classic T(n) = 2T(n-1) + 1 yields exponential O(2^n) solution.
